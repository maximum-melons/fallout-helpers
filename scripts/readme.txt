To run any custom script in Fallout 4 over console:

1. copy the script into your game folder (the folder with Fallout4.exe, e.g. on Windows through Steam: C:\Program Files (x86)\Steam\steamapps\common\Fallout 4):
2. launch Fallout 4
3. open console via ` key
4. run the command 'bat <name>' where <name> is the filename of your script without the extension (i.e. 'bat materials')
